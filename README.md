# Мои Учебные Задачи по Java

Этот репозиторий содержит решения различных учебных задач по программированию на Java. Каждая задача находится в отдельном пакете и сопровождается собственным подробным описанием.

---

## Список Задач:

### 1. [Класс "Матрица" (Matrix)](src/com/rimanets/javamatrix/README.md)

* **Описание:** Реализация класса для работы с матрицами, включающая конструктор, методы для установки значений и сложения матриц, а также обработку исключений. Этот класс демонстрирует основы объектно-ориентированного программирования и работу с двумерными массивами.
* **Расположение кода:** `src/com/rimanets/javamatrix/`

---

## Как Запустить Проект:

1.  **Клонируйте репозиторий:**
    ```bash
    git clone [https://github.com/rimanets/javaMatrixProject.git](https://github.com/rimanets/javaMatrixProject.git)
    cd javaMatrixProject
    ```
2.  **Откройте проект в IDE:** Импортируйте проект в IntelliJ IDEA, Eclipse или другую вашу любимую IDE как Maven/Gradle проект (если используете) или как обычный Java-проект.
3.  **Запустите `Main.java`:** Для запуска тестов и демонстрации работы класса `Matrix`, найдите файл `src/main/java/com/rimanets/javamatrix/Main.java` и запустите его метод `main`.
